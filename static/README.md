# static

**このディレクトリは必須ではありません。必要なければ削除することができます。**

このディレクトリには静的ファイルを入れます。
このディレクトリ内のファイルはいずれも / に配置されます。

例: `/static/robots.txt` は `/robots.txt` に配置されます。
